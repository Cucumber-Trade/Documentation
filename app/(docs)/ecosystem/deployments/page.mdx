## Testnet Faucet

Request testnet tokens for development:

```bash
cucumber faucet --network sepolia --amount 10000
```

Or visit [faucet.cucumbertrade.com](https://faucet.cucumbertrade.com) to claim tokens manually.

## Verification

All contracts are verified on Etherscan with matching source code. To verify locally:

```bash
cucumber verify --network mainnet --contract AgentRegistry
```
