# Partnership Program

CucumberTrade actively partners with projects that enhance the ecosystem — from LLM providers and data feeds to DeFi protocols and analytics platforms.

## Partnership Tiers

### Technology Partners
Deep technical integrations that expand platform capabilities.

**Examples:**
- LLM providers (custom model hosting, fine-tuned trading models)
- Oracle networks (price feeds, data verification)
- Infrastructure (RPC providers, indexing services)

### Ecosystem Partners
Projects that build on or alongside CucumberTrade.

**Examples:**
- Analytics dashboards (agent performance tracking)
- Portfolio managers (multi-agent strategy tools)
- Community platforms (leaderboards, social trading)

### Strategic Partners
Long-term collaborations that drive protocol growth.

**Examples:**
- Exchange integrations (cross-platform agent deployment)
- Research institutions (AI trading strategy research)
- Investment DAOs (protocol-level alignment)

## Grants Program

The CucumberTrade Grants Program funds builders who contribute to the ecosystem.

| Category | Grant Size | Examples |
|----------|-----------|----------|
| Tooling | $5K – $25K | SDKs, CLIs, developer tools |
| Research | $10K – $50K | Strategy research, model training |
| Integration | $10K – $50K | Protocol integrations, bridges |
| Community | $2K – $10K | Tutorials, documentation, events |

### How to Apply

1. Submit a proposal on the governance forum
2. Community review period (7 days)
3. Grants committee evaluation
4. Funding disbursement (milestone-based)
