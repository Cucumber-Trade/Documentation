# Contact & Community

Connect with the CucumberTrade team and community across our official channels.

## Community Channels

### Discord
Our primary community hub for real-time discussion, support, and announcements.
- General discussion and Q&A
- Developer channels for technical help
- Strategy sharing and agent showcases
- Governance proposals and debates

### X (Twitter)
Follow [@CucumberTrade](https://x.com/CucumberTrade) for announcements, updates, and ecosystem highlights.

### GitHub
Browse source code, report issues, and contribute to the open-source ecosystem at [github.com/Cucumber-Trade](https://github.com/Cucumber-Trade).

## Support

### Community Support
For general questions, use the **#support** channel in Discord. Community members and team moderators are available to help.

### Developer Support
For technical integration questions, use the **#dev-support** channel in Discord or open an issue on GitHub.

### Enterprise Support
For partnership inquiries, custom integrations, or enterprise-grade support, contact us at hello@cucumber.trade.

### Security Reports
For responsible vulnerability disclosure, email hello@cucumber.trade. See our [Bug Bounty Program](/ecosystem/security) for reward details.

## Contributing

CucumberTrade welcomes community contributions:

- **Documentation** — Improve or translate docs via GitHub PRs
- **SDKs & Tools** — Build developer tools and apply for ecosystem grants
- **Strategy Templates** — Share strategy templates with the community
- **Bug Reports** — Report issues through GitHub or Discord

See the [Partnership Program](/ecosystem/partners) for information about grants and funding for ecosystem projects.
